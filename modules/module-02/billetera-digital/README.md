# 💳 Billetera Digital

## 🧱 Stack
HTML5 + CSS3 + ES6 + Bootstrap 5.3.8 + jQuery 4

## 🐳 Ejecución
El proyecto se ejecuta con Docker usando Nginx como servidor estático local.

## ⚙️ Versión activa
La versión servida se define en `docker-compose.yml` mediante la variable `WEB_03` que apunta a la carpeta del proyecto.

## 📌 Versiones disponibles
WEB_01=01-maquetacion  
WEB_02=02-javascript  
WEB_03=03-jquery  

## 🚀 Levantar entorno
./start.sh

## 🛑 Detener entorno
./stop.sh

## 🌐 Acceso
http://localhost:8080
