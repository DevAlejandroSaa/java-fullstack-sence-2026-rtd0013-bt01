# 📦 README

## 📝 Descripción

Aplicación web estática desarrollada como parte de la evaluación final del Módulo 2.

## 🛠️ Stack Tecnológico

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript ES6
* 🅱️ Bootstrap 5.3.8
* 💻 jQuery 4

## 🐳 Contenedor Docker

La aplicación se ejecuta utilizando la imagen:

```text
nginx:alpine
```

## ▶️ Iniciar el proyecto

```bash
./start.sh
```

## ⏹️ Detener el proyecto

```bash
./stop.sh
```
